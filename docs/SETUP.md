# Setup

1. Install Flutter stable and Dart SDK.
2. Run `flutter pub get`.
3. Configure Firebase for Android/iOS/Web using FlutterFire CLI.
4. Add platform config files (`google-services.json`, `GoogleService-Info.plist`, web config).
5. Run app with `flutter run`.
6. See `docs/FIREBASE_SETUP.md` for web `--dart-define` values.
